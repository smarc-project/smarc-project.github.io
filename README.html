smarc_docs¶

Documentation for the SMaRC project software repositories

Generating the documentation¶

Dependencies¶

Run sudo pip install sphinx sphinx_rtd_theme to get the necessary dependencies.

Commands¶

Go into the root of this repo. Run make html. Copy all contents of the html folder into the root of https://github.com/smarc-project/smarc-project.github.io . Commit the changes to that repo and push or open a pull request to get the changes up to https://smarc-project.github.io/ .
